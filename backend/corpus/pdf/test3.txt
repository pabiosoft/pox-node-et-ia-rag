PDF Test 3 - Science des Données

La science des données est un domaine interdisciplinaire qui utilise des méthodes 
scientifiques, des processus, des algorithmes et des systèmes pour extraire des 
connaissances et des insights à partir de données structurées et non structurées.

Les étapes clés du processus:
1. Collecte des données
2. Nettoyage et préparation
3. Analyse exploratoire
4. Modélisation (Machine Learning)
5. Visualisation et reporting

Les outils populaires:
- Python (Pandas, NumPy, Scikit-learn)
- R (dplyr, ggplot2)
- SQL pour les bases de données
- Tableau/Power BI pour la visualisation

Les data scientists jouent un rôle crucial dans la prise de décision basée sur les données.
